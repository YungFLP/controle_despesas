Controle de Despesas

Projeto simples em Python para registrar despesas do dia a dia usando SQLite.

A ideia do projeto é permitir adicionar, listar e excluir despesas através de um menu no terminal.

Funcionalidades

Adicionar despesa

Listar despesas

Excluir despesa

Armazenamento em banco de dados SQLite

Tecnologias utilizadas

Python

SQLite

Git / GitHub

Como executar

Clone o repositório:

git clone https://github.com/YungFLP/controle_despesas

Entre na pasta do projeto:

cd controle_despesas

Execute o programa:

python app.py
